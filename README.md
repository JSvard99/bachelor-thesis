# Overview
This is my bachelors thesis in collaboration with one other student. The purpose of this work was to integrate the static code analysis program SpotBugs with AI to filter out false positives that are often present in these types of tools. The tool is written with object-oriented Python and has a GUI created with Tkinter. 

The thesis is also available here: https://www.diva-portal.org/smash/record.jsf?pid=diva2%3A1995525&dswid=2433 

The source code for the tool is available here: https://bitbucket.org/jsvard/thesis-tool/src/main/
